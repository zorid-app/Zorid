# Project Guidelines

Everytime a coding session is completed and finalized, add the files you changed, and commit them with a clear and descriptive commit message. DO NOT include omx lore. Just plain commit. 

After you commit, create a merge request plain text file in the .agent-context/merge-requests directory. File name should be your branch name. The file should contain the commit message.

When the user said it has screenshots, the screenshot is located at the .agent-context/screenshots directory. The screenshot might have timestamps on it. Always open the latest ones unless otherwise specified.
